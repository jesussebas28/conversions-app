# conversions-app

Una herramienta de cálculo de imposición y distribución de piezas sobre pliegos de impresión. Diseñada para optimizar el aprovechamiento de material en entornos de artes gráficas.

## 🛠️ Características Principales

- **Cálculo Dinámico**: Determina instantáneamente cuántas piezas caben en un contenedor específico (Tabloide, Carta, Pliego Largo).
- **Simulación Visual**: Genera un canvas a escala que representa la cuadrícula de distribución real.
- **Rotación de Pieza**: Permite forzar la rotación de 90° de las imágenes para evaluar mejores aprovechamientos de área.
- **Reporte de Área**: Indica el área total utilizada frente al área disponible del papel.

## ⚙️ Instalación

### clonar proyecto
```bash
git clone https://github.com/jesussebas28/conversions-app.git;
cd conversions-app
```

### Instalar dependencias

```bash
pnpm install
```

### Iniciar servidor de desarrollo

```bash
pnpm dev
```

### Construir para producción
```bash
pnpm build
```
